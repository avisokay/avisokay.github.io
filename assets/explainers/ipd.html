ü¶è Inference on Predicted Data (IPD)

The problem: a researcher is interested in studying an outcome, Y, which is difficult to measure due to practical constraints such as time or cost. But they do have access to relatively cheap predictions of Y. They hypothesize that Y is associated with X, a set of features which are easier to measure. Their goal is to estimate a parameter of scientific interest, Œ∏, which describes the relationship between X and Y. How can the researcher attain valid estimates of Œ∏ using mostly predicted outcomes of Y?

For a more extensive overview, please see Hoffman et al. 2024 and the code respository for the `ipd` package.
